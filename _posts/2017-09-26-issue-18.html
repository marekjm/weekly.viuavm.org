This was another slow week, and it resulted in almost no newsworthy developments.

{{ page.sections[0].name }}

To disambiguate function names from labels, function names are now required to include an arity specifier at the end. The arity may be undefined for functions which require it.

{% highlight lua %} .function: example/0 -- a function with specified arity of 0 (a nullary function) .function: example/1 -- a function with specified arity of 1 (an unary function) .function: example/ -- a function with undefined arity {% endhighlight %}

{{ page.sections[1].name }}

Invalid operands for are now reported for jump instructions.

{% highlight lua %} jump +1 -- OK, forward jump jump -1 -- OK, backward jump jump here -- OK, jump to a marker jump "Hello" -- jumps to strings (or ints, or bits, etc.) are not allowed {% endhighlight %}
