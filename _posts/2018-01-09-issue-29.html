New documentation is being written. So far it does not cover all instructions and is not as in-depth as it should be, but it is only a matter of time before it will be.

{{ page.sections[0].name }}

New documentation is written in a form of a single-file document covering everything there is to cover about the VM. Available instructions, syntax of the assembly language, tools that are shipped in a standard distribution, programming techniques, etc.

There are three versions of the manual (all with the same content):

  • plaintext version
  • colorised HTML version (dark background, light text)
  • browser-default HTML version (whatever your browser-default colors happen to be

All versions have exactly the same content, and are rendered in exactly the same way; except for the colors and links from ToC to sections in the HTML versions.
